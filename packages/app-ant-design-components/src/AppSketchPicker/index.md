## API

| 参数     | 说明                     | 类型                      | 默认值   |
| -------- | ------------------------ | ------------------------- | -------- |
| color    | 展示取色器的颜色         | `string`                  | `主题色` |
| onChange | 改变取色器颜色的回调函数 | `(color: string) => void` | -        |
