# AppSetting 设置组件

该组件实现是统一的网站小图标设置网站通用数据的封装，比如系统级别的主题色、多语言、字体大小、组件的圆角等，目前暂时支持内部组件自己维护状态，也可以通过传入组件进行渲染。

目前设计点

- 主题色可变
  - 多语言的配置
- 基于侧拉框的封装
- 基于 ts 进行封装
